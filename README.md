# Blog App Next JS 14
this app is for read blogs and upload blogs for other users with Better UI and Latest Tech Next JS 14.
## Features
- account creation via Github and Email
- read uploaded Blogs
- Create New Blog
## Tech Stack
- Next JS 14
- Mongo DB
- other tools
